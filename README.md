# AltoTrail

Public documentation and portfolio material for AltoTrail, a multilingual
job-search and labour-market exploration product.

AltoTrail is an independently developed multilingual job-search and
labour-market exploration product in active development.

The product goal is to help people explore jobs, occupations, skills, and
labour-market context across languages. AltoTrail is designed as a public-facing
product with careful boundaries between open information, private operations,
and user-specific features.

## Repository scope

This repository is a public documentation and portfolio repository only. It is
not the production application repository.

Included here:

- Product overview.
- High-level architecture principles.
- Public data and privacy principles.
- SEO, internationalization, and accessibility principles.
- Security, trademark, and notice documentation.

Not included here:

- Production source code.
- Private implementation details.
- Deployment or scheduler configuration.
- Operational runbooks, logs, snapshots, or reports.
- Job alert internals.
- Data ingestion, cache implementation, generated datasets, or prompts.
- Credentials, tokens, analytics identifiers, private URLs, or personal data.

## Product status

AltoTrail is in active development and may be operated commercially, including
advertising, partnerships, and paid product development.

No rights are granted to reuse the AltoTrail brand, product identity, private
implementation, datasets, production systems, or commercial internals unless
explicitly stated in writing.
