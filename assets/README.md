# Assets

This directory is reserved for future public portfolio assets, such as reviewed
screenshots or diagrams.

Do not add assets that contain:

- Personal data.
- Private URLs.
- Credentials, tokens, or identifiers.
- Analytics identifiers.
- Operational dashboards, logs, reports, or runbooks.
- Production configuration.
- Private implementation details.

Any screenshot or diagram should be reviewed before publication to confirm that
it is safe for a public documentation repository.
