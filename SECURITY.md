# Security Policy

## Scope

This repository contains public documentation and portfolio material for
AltoTrail. It does not contain production source code, deployment configuration,
credentials, private datasets, operational runbooks, or user records.

## Reporting security issues

Use the public contact information available on altotrail.com to report a
security concern.

Please do not publish sensitive details publicly before the issue has been
reviewed. Reports should avoid including unnecessary personal data, credentials,
tokens, private endpoint details, or exploit instructions.

## Out of scope for this repository

The following are not available through this public repository:

- Production system access.
- Private source code.
- Operational runbooks or infrastructure details.
- User data or subscription data.
- Internal datasets, generated data, prompts, or delivery systems.

## Documentation issues

If you find public documentation that appears to expose private operational
details, personal data, credentials, or sensitive identifiers, please report it
through the public contact information on altotrail.com.
